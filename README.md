# RemNote_AutoHotkey
感谢【AutoHotkey(ahk) | 中文社区】的管理员“混沌+AHK2”和“AHK1-僵尸”。

```
 :*T://:://   ;打开菜单
 :*T:[[::[[   ;使用双链
 :*T:$$::$$   ;输入公式
 :*T:!!::!!   ;寻找日期
 :*T:##::##   ;输入Tag标签
 :*T:%%::%%   ;输入emoji表情
 :*T:``::``   ;输入引述
```

需要在[Settings]中
  [Keyboard Shortcuts]
  
打开  [Double Slash to Trigger the Command Menu]
